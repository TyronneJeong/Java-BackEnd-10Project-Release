Reflection
- 힙 영역에 로드되어 있는 클래스 타입의 객체를 이용하여 필드/메소드/생성자를 접근 제어여부와 관계없이 사용가능토록 도와주는 API
- 컴파일 시점이 아닌 런타임 시점에서 클래스 정보를 추출 할 수 있으며, 주로 프레임워크 또는 라이브러리 개발시 이용됨.
- https://www.baeldung.com/reflectinos-libaray


Reflection 기술을 이용하는 프레임워크 또는 라이브러리
- Spring 프레임워크 (DI)
- Test 프레임워크 (Junit)
- JSON serialization/Deserialization 라이브러리 (Jackson)
